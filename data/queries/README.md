# Query Files

This directory stores normalized query files using schema version 1.0. Warsaw, Bellevue, and Beach are ready single-video datasets with dataset-default video ids. QVHighlights contains the supplied 20 query texts, but the source file does not include per-query video ids, so it is marked `missing_video_id` in `results/paper/query_manifest.csv`. See `QVHIGHLIGHTS_MAPPING_REPORT.md` for the mapping audit.
