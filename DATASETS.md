# Datasets

No raw video dataset is included. Do not commit LAVA, LOVO, QVHighlights, Beach, Bellevue, Warsaw, or other raw videos unless licenses explicitly allow redistribution.

| Dataset | Expected directory | Included | Notes |
|---|---|---|---|
| LAVA | `datasets/LAVA/` | no | source/license/checksum must be filled by authors |
| LOVO-style videos | `datasets/LOVO/` | no | external data |
| QVHighlights | `datasets/QVHighlights/` | no | external benchmark |
| Beach/Bellevue/Warsaw | `datasets/<name>/` | no | local experimental data; verify license |
